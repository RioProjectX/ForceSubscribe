### Deploy in vps
- Clone this repo
```
git clone https://github.com/Jigarvarma2005/ForceSub_Bot
```
- Change directory
```
cd ForceSub_Bot
```
- Install requirements
```
pip3 install -r requirements.txt
```
- setup configuration
```
Open Config.py in text Editor and set values
```
- Start the bot
```
python3 bot.py
```
## Configs:
- Read `Config.py` to know all configs
